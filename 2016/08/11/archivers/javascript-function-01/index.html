解析： foo函数中返回值除非是对象，否则就将this作为返回值返回，this会被初始化为默认的一个空对象，并且this的原型是foo.prototype，该空对象的b属性会赋值为100，然后整个对象会被作为返回值返回并且忽略return的返回值。
