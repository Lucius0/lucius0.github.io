git clone时遇到could not read from remote repository

发表于 2017-01-07 | 分类于 tools | | 浏览 次

git clone 时遇到如下的错误。

问题原因

缺少 known_hosts 文件, 而且必须生成 github.com 的ip执行内容.

只要执行命令ssh git@github.com，这样在.ssh目录就有三个文件了，即id_rsa，id_rsa.pub，known_hosts
